# Быстрый старт: AI-Dieta с онлайн Food Recognition

## 🚀 Что было сделано

Мы успешно заменили локальную нейросетевую модель YOLOv8 на бесплатные онлайн API для распознавания еды. Это решает проблему с большим размером Docker образа (с >12ГБ до ~100-200МБ).

## 📋 Пошаговая инструкция

### Шаг 1: Регистрация в API провайдере

1. **Зарегистрируйтесь в Edamam API** (рекомендуется):
   ```bash
   # Перейдите по ссылке: https://developer.edamam.com/
   # Зарегистрируйте аккаунт
   # Создайте новое приложение
   # Получите APP_ID и APP_KEY
   ```

2. **Альтернативные провайдеры** (если Edamam не подходит):
   - Google Cloud Vision: https://cloud.google.com/vision
   - Azure Computer Vision: https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/
   - Clarifai: https://www.clarifai.com/

### Шаг 2: Настройка переменных окружения

```bash
# Установите переменные окружения (временно)
export EDAMAM_APP_ID=ваш_app_id_здесь
export EDAMAM_APP_KEY=ваш_app_key_здесь

# Для постоянной настройки добавьте в ~/.bashrc
echo 'export EDAMAM_APP_ID=ваш_app_id_здесь' >> ~/.bashrc
echo 'export EDAMAM_APP_KEY=ваш_app_key_здесь' >> ~/.bashrc
source ~/.bashrc
```

### Шаг 3: Проверка API

```bash
# Перейдите в директорию backend
cd /home/user/Рабочий стол/App/стол/App/backend

# Запустите тест API
python3 test_edamam_api.py
```

### Шаг 4: Запуск приложения

```bash
# Установите зависимости
pip3 install -r requirements.txt

# Запустите сервер
uvicorn app.main:app --reload

# Откройте документацию API
# http://localhost:8000/docs
```

## 🍎 Как использовать Food Recognition API

### 1. Загрузка изображения

```bash
# Пример curl запроса
curl -X POST "http://localhost:8000/api/vision/recognize" \
  -H "Authorization: Bearer ваш_токен" \
  -F "image=@путь/к/изображению.jpg"
```

### 2. Пример ответа

```json
{
  "recognized_foods": [
    {
      "name": "Apple",
      "calories": 95,
      "confidence": 0.9
    },
    {
      "name": "Banana",
      "calories": 105,
      "confidence": 0.85
    }
  ]
}
```

### 3. Получение информации о доступных API

```bash
curl "http://localhost:8000/api/vision/available-apis"
```

## 🔧 Технические детали

### Поддерживаемые API

1. **Edamam Food Detection API** (основной)
   - 1000 запросов/месяц бесплатно
   - Специализирован на распознавании еды
   - Возвращает калории и нутриенты

2. **Fallback-режим**
   - Автоматически включается при отсутствии API ключей
   - Возвращает список常见ых продуктов с приблизительной калорийностью
   - Гарантирует работу приложения в любом случае

### Преимущества решения

✅ **Малый размер Docker образа** (100-200МБ вместо >12ГБ)  
✅ **Высокая точность распознавания** (профессиональные API)  
✅ **Низкие требования к ресурсам** (не нужен GPU)  
✅ **Автоматическое обновление** (API обновляются провайдерами)  
✅ **Бесплатное использование** (достаточно для большинства приложений)  

## 🚨 Важные моменты

1. **API ключи** - необходимо зарегистрироваться и получить ключи
2. **Интернет-соединение** - требуется для работы онлайн API
3. **Ограничения бесплатного тарифа** - 1000 запросов/месяц для Edamam
4. **Fallback-режим** - работает без интернета, но с ограниченной точностью

## 🆘 Тroubleshooting

### API ключи не работают
```bash
# Проверьте, что переменные установлены
echo $EDAMAM_APP_ID
echo $EDAMAM_APP_KEY

# Перезапустите терминал или выполните
source ~/.bashrc
```

### Ошибка подключения к API
```bash
# Проверьте интернет-соединение
ping api.edamam.com

# Проверьте API ключи на сайте Edamam
```

### Docker образ все еще большой
```bash
# Убедитесь, что .dockerignore работает
docker build --no-cache -t ai-dieta-backend .
docker images | grep ai-dieta
```
