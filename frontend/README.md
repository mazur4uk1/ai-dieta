# AI-Dieta Frontend

Современная админ-панель для управления приложением AI-Dieta с минималистичным дизайном и 3D анимациями.

## 🎨 Дизайн

- **Цветовая палитра:**
  - Jungle Teal `#6b9080` - основные элементы
  - Muted Teal `#a4c3b2` - акценты
  - Frozen Water `#cce3de` - фоновые элементы
  - Azure Mist `#eaf4f4` - легкие фоны
  - Mint Cream `#f6fff8` - основной фон

- **Особенности:**
  - Минималистичный дизайн
  - Плавные 3D анимации (Framer Motion)
  - Адаптивный интерфейс
  - Темная/светлая тема

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

## 🏗️ Стек технологий

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **React Query** - управление состоянием API
- **Recharts** - графики и диаграммы
- **Lucide React** - иконки

## 📁 Структура проекта

```
src/
├── app/                 # Next.js App Router
│   ├── layout.tsx      # Корневой layout
│   ├── page.tsx        # Главная страница
│   └── globals.css     # Глобальные стили
├── components/         # React компоненты
│   ├── dashboard.tsx   # Главный дашборд
│   ├── stats-chart.tsx # Графики статистики
│   └── ui/            # UI компоненты
├── hooks/             # React хуки
│   └── use-api.ts     # API хуки
├── lib/               # Утилиты
│   ├── api.ts         # API клиент
│   └── query-provider.tsx # React Query провайдер
└── types/             # TypeScript типы
    └── index.ts       # Определения типов
```

## 🔧 Конфигурация

Создайте `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1
```
